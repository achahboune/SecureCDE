# Aegis Brand System

Aegis is a compliance-first security brand.

This document defines the official design language.

---

| Usage | Color | Hex |
|-----|-----|-----|
| Background | Aegis Navy | `#070B1A` |
| Surface / Cards | Midnight | `#0B122A` |
| Borders / Stroke | Steel | `#1B2A4A` |
| Primary Text | Ice White | `#EAF0FF` |
| Secondary Text | Muted Blue | `#A9B7D0` |
| Primary CTA | Aegis Cyan | `#00E5FF` |
| Secondary | Aegis Blue | `#2F6BFF` |
| Success | Green | `#2EE59D` |
| Warning | Amber | `#FFB84D` |
| Danger | Red | `#FF4D6D` |
